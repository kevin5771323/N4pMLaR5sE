# 家教管理系统 java290

## 项目概述

本项目是基于SpringBoot和Vue3的家教管理系统，采用前后端分离的技术架构，提供管理员、教师和用户三种角色的功能模块。前端使用Vue3、Element Plus UI组件库、Pinia状态管理、VueRouter4路由管理，后端采用SpringBoot、Mybatis-Plus和MySQL技术栈。

## 技术栈

- 后端：SpringBoot、Mybatis-Plus、MySQL
- 前端：Vue3、Element-Plus、Vue-Router、Pinia、Vite

## 功能模块

### 系统角色

#### 管理员（Admin）

- 管理员、用户、教师管理
- 预约信息管理
- 课程管理
  - 新增编辑课程信息
  - 查看课程留言
  - 回复留言
  - 课程收藏管理
- 论坛管理
  - 查看详情
  - 查看论坛回复
  - 删除帖子
- 基础数据管理
  - 教师类型、课程类型、公告类型
- 轮播图管理

#### 教师（Teacher）

- 登录、注册、修改个人信息
- 教师管理
  - 查看教师评价
  - 回复评价
  - 教师收藏管理
- 预约信息管理
  - 查看预约详情
  - 审核预约信息
- 课程管理
  - 查看课程信息
  - 查看课程留言信息
- 论坛管理
  - 查看详情
  - 查看论坛回复
  - 删除帖子

#### 用户（User）

- 登录、注册、修改个人信息、充值
- 查看课程信息
- 教师信息收藏、评价、预约
- 预约信息管理
  - 查看审核状态
  - 审核回复
  - 支付、取消
- 查看教师收藏、教师评价、课程收藏、课程留言信息
- 论坛功能
  - 查看帖子
  - 评论帖子
  - 发布帖子

## 运行环境

- 操作系统: Win10/11
- 开发工具: IDEA IU 2022.3.3
- 开发语言: Java (JDK 8u371)
- 项目管理工具: Apache Maven 3.9.4
- 数据库: MySQL 8.0
- 前端环境:
  - Node.js: v18.20.0
  - 构建工具: Vite

## 部署步骤

1. 执行目录中的sql文件（可选）
2. 解压项目压缩包
